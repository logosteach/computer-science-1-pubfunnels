# Unit 10 - Testing Your Code

This unit contains the following lessons:

- The `assert` statement
- `unittest`
- `pytest`

## Lesson 1 Objectives

By the end of the lesson, students will be able to:

- define the purpose and usage of the Python `assert` keyword.
- explain when the `assert` keyword is useful and when it should not be used.
- identify when a block of code containing an `assert` statement will raise an `AssertionError`.
- write code using comparison assertions.
- write code using membership assertions.
- write code using identity assertions.
- use assertions as inline documentation in their code.
- test their own programs using assertions.
- articulate how the practice of using assertions to verify truth in code reflects the biblical principle of testing all things and holding fast to what is good (1 Thessalonians 5:21).

## Lesson 2 Objectives

By the end of the lesson, students will be able to:

- import the `unittest` module into their programs.
- create a test class that inherits from `unittest.TestCase`.
- write test methods named with the `test_` prefix inside a test class.
- use assertion methods provided by `unittest.TestCase` in their test methods.
- run unit tests using `unittest` both from the terminal and from within Python code.
- articulate how the LORD righteously tests those who take refuge in Him and the purposes behind His testing (Psalm 11:5; Deuteronomy 8:2).

## Lesson 3 Objectives

By the end of the lesson, students will be able to:

- install the `pytest` package into a virtual environment.
- write test functions for `pytest` using Python's built-in `assert` keyword.
- describe the advantages of `pytest` compared to `unittest`.
- explain the concept of boilerplate code.
- use the `@pytest.fixture` decorator in their test programs.
- use the `@pytest.mark.parametrize` decorator to create parameterized tests.
- write parameterized test functions using `@pytest.mark.parametrize`.
- read the following verses (John 17:17; 2 Corinthians 6:17; 2 Timothy 2:21) and develop one or more parallels between the concept of a Python virtual environment and how the LORD sanctifies, sets apart, and equips believers—creating a Christ centered, and Christ focused walk in their lives, dying to self daily, so they can be useful vessels for His good purposes and plans.
