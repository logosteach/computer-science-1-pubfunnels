# Unit 2 - Foundational Elements

This unit contains the following lessons:

- Lesson 1: Order of Operations in Python
- Lesson 2: Variables
- Lesson 3: Data Types
- Lesson 4: Printing and Escape Characters
- Lesson 5: F-Strings
- Lesson 6: Division in Python /, //, and %
- Lesson 7: int(), float(), and str()
- Lesson 8: Commenting in Code (Inline and Paragraph)
- Lesson 9: Slicing Strings and Immutability of Strings
- Lesson 10: The input() function

## Lesson 1 Objectives

By the end of the lesson, students will be able to:

- explain the order of operations (PEMDAS/BODMAS) and how Python evaluates mathematical expressions.
- use parentheses to control the order of operations in Python.
- evaluate complex arithmetic expressions correctly in Python.
- identify common mistakes related to operator precedence.
- reflect on how the order of operations in Python parallels Jesus’ words to His disciples: “I still have many things to say to you, but you cannot bear them now” (John 16:12). Just as some operations must wait until others are completed, we often need to wait on the Lord’s timing before we are ready for the challenges or blessings He has for us.

## Lesson 2 Objectives

By the end of the lesson, students will be able to:

- declare and assign values to variables in Python.
- understand variable naming rules and best practices.
- reassign values to variables and explain the concept of mutability.
- use variables in mathematical expressions and output.
- contrast the mutable nature of variables in Python (which can change value) with the immutable promises of God, which are “Yes” and “Amen” in Christ Jesus and never change, giving us a firm foundation we can always rely on (2 Corinthians 1:20).

## Lesson 3 Objectives

By the end of the lesson, students will be able to:

- identify and describe the main data types in Python (int, float, str, bool).
- use the `type()` function to check the data type of a value or variable.
- explain the differences between data types and why they matter.
- convert between basic data types when necessary.
- recognize that just as different data types in Python require different operations, the gospel is light that exposes darkness, yet Jesus approached people in different ways. While people are not data types, knowing who we are speaking to is important, as Jesus demonstrated when engaging Nicodemus (John 3), the Samaritan woman (John 4), and the rich young ruler (Matthew 19).

## Lesson 4 Objectives

By the end of the lesson, students will be able to:

- use the `print()` function with multiple arguments and separators.
- apply escape characters such as `\n`, `\t`, `\\`, and `\"` effectively.
- control the formatting and layout of program output.
- create clean, readable console output using escape sequences.
- recognize that just as we use escape characters in `print()` statements to create clear and readable output, God has made the gospel clear and understandable as light that shines in the darkness. While some passages may be difficult, the message of salvation is vivid and direct, calling us to look to Christ and Christ alone (2 Corinthians 4:6; John 8:12).

## Lesson 5 Objectives

By the end of the lesson, students will be able to:

- explain the advantages of f-strings over older string formatting methods.
- create and use f-strings to embed variables and expressions inside strings.
- format numbers, dates, and text using f-string formatting options.
- write clear and readable code using f-strings for dynamic output.
- recognize that just as f-strings allow us to inject variable values into strings so we can clearly see their content, applying the teachings of Christ to our lives makes Him more visible to us. What was once hidden becomes clearer, and we grow in our understanding and appreciation of the true value of belonging to God in Christ Jesus (Colossians 2:2-3; Philippians 3:8).

## Lesson 6 Objectives

By the end of the lesson, students will be able to:

- differentiate between the three division operators in Python: `/`, `//`, and `%`.
- use `/` for floating-point division, `//` for floor division, and `%` for modulo (remainder).
- apply these operators to solve real-world problems (e.g., splitting items, finding remainders).
- predict the output of expressions involving different division types.
- reflect on how learning to properly use the different division operators in Python parallels the biblical call to “rightly divide the word of truth.” Just as we must carefully choose the right operation for accurate results, we are called to diligently study and rightly handle God’s Word so we may present Christ clearly in the gospel (2 Timothy 2:15).

## Lesson 7 Objectives

By the end of the lesson, students will be able to:

- use the `int()`, `float()`, and `str()` functions to convert between data types.
- explain when and why type conversion is necessary.
- handle user input safely by converting strings to numbers.
- avoid common errors caused by incorrect data types.
- recognize that just as a string like "37" cannot transform itself into the integer 37 without the `int()` function, we cannot lift ourselves from spiritual death to life on our own. We are completely dependent on the transforming work of Christ and faith in Him to make us alive (Ephesians 2:1-5).

## Lesson 8 Objectives

By the end of the lesson, students will be able to:

- write inline comments using the `#` symbol.
- create multi-line (paragraph) comments using triple quotes (`"""` or `'''`).
- explain best practices for writing clear and useful comments.
- use comments effectively to document code logic and purpose.
- recognize that just as we use comments in code to clarify sections and prevent misunderstanding, the Apostle Paul sometimes inserts clarifying explanations in his letters. In 1 Corinthians 1:14-16, he adds a block-like clarification so his readers would not draw the wrong conclusion about his ministry.

## Lesson 9 Objectives

By the end of the lesson, students will be able to:

- slice strings using index notation (`[start:end:step]`).
- understand and demonstrate string immutability in Python.
- explain what happens when attempting to modify a string directly.
- use string slicing to extract and manipulate portions of text.
- describe how the immutability of strings in Python is similar to, yet far different from, the perfect immutability of God. While strings are limited and restrictive in their immutability, God’s character, promises, and nature never change — He is the same yesterday, today, and forever (Malachi 3:6; Hebrews 13:8; James 1:17).

## Lesson 10 Objectives

By the end of the lesson, students will be able to:

- use the `input()` function to get data from the user.
- store and process user input in variables.
- convert user input from string to appropriate data types.
- create simple interactive programs using `input()` and `print()`.
- recognize that just as the `input()` function allows us to enter values into our programs so they can be processed and used, God invites us to bring our needs to Him through prayer. Even though He already knows what we need, He still calls us to ask with thanksgiving because we are in a personal relationship with Him through Christ Jesus (Philippians 4:6-7, 19; Matthew 6:8).
