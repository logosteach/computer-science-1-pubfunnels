# Unit 11 - Data Science (A Gentle Introduction)

This unit contains the following lessons:

- Lesson 1: What is Data Science and the Data Science Life Cycle?
- Lesson 2: Introduction to NumPy
- Lesson 3: Introduction to Pandas
- Lesson 4: Data Visualization with Matplotlib
- Lesson 5: Putting It All Together - Creating and Visualizing Data

## Lesson 1 Objectives

By the end of the lesson, students will be able to:

- explain what Data Science is and why it is important in today’s world.
- describe the main stages of the Data Science Life Cycle (DSLC).
- understand how Python is widely used in data science and analysis.
- recognize that this unit is a light introduction designed to open the door to future learning.
- reflect on how God has given humanity the ability to explore, understand, and steward creation through careful observation and analysis (Genesis 1:28; Psalm 111:2).

## Lesson 2 Objectives

By the end of the lesson, students will be able to:

- explain what NumPy is and why it is foundational for scientific computing and data science.
- create 1D and 2D NumPy arrays using `np.array()`, `np.zeros()`, and `np.ones()`.
- perform basic mathematical operations on arrays (addition, multiplication, etc.).
- create and multiply matrices using NumPy.
- generate random data using `numpy.random`.
- understand how NumPy enables fast and efficient numerical computations in Python.

## Lesson 3 Objectives

By the end of the lesson, students will be able to:

- explain what Pandas is and its main data structures (`Series` and `DataFrame`).
- create a Pandas DataFrame from a dictionary or NumPy array.
- perform basic data exploration using methods like `.head()`, `.describe()`, and `.info()`.
- select columns and filter rows in a DataFrame.
- understand why Pandas is one of the most popular tools for data analysis in Python.

## Lesson 4 Objectives

By the end of the lesson, students will be able to:

- import and use `matplotlib.pyplot` for creating charts and graphs.
- create line plots, scatter plots, and histograms using Matplotlib.
- customize plots with titles, axis labels, and colors.
- generate a random 2D dataset using NumPy and visualize it with Matplotlib.
- create histograms to explore the distribution of data.

## Lesson 5 Objectives

By the end of the lesson, students will be able to:

- combine NumPy, Pandas, and Matplotlib in a single workflow.
- create 1D and 2D arrays of zeros and ones using NumPy.
- perform matrix multiplication using NumPy.
- generate random datasets and create meaningful visualizations.
- reflect on what they have learned and how these powerful tools can open doors for future studies or careers.
- recognize that while this unit only opens the door to data science, the habits of careful observation, clear thinking, and honest analysis reflect godly stewardship (Proverbs 24:3-4; Colossians 3:23).
