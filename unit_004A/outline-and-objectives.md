# Unit 4A - Logic Flow and Conditional Statements

This unit contains the following lessons:

- Lesson 1: Boolean Values and Variables
- Lesson 2: Comparator Operators
- Lesson 3A: Compound Conditional Expressions
- Lesson 3B: Short-Circuit Evaluation
- Lesson 4A: IF, ELIF, and ELSE blocks
- Lesson 4B: Nested IF Blocks
- Lesson 5: Truthy and Falsy Values
- Lesson 6: Common Pitfalls and Best Practices
- Lesson 7: Ternary Conditional Expressions
- Lesson 8: Practical, Real World Problems

## Lesson 1 Objectives

By the end of the lesson, students will be able to:

- explain what Boolean values are and how they are used in Python (`True` and `False`).
- create and assign Boolean values to variables.
- use Boolean variables in conditional logic.
- understand that Boolean values form the foundation of all decision-making in programming.
- reflect on how God has given us the ability to discern truth from falsehood, just as Boolean values allow our programs to make clear decisions (John 8:32; 1 Kings 18:21).

## Lesson 2 Objectives

By the end of the lesson, students will be able to:

- use the six comparator operators in Python: `==`, `!=`, `>`, `<`, `>=`, `<=`.
- write conditions that compare numbers, strings, and variables.
- predict the Boolean result of comparator expressions.
- understand how comparators are used to make decisions in code.
- recognize that just as we compare values in code, Scripture calls us to test and discern what is good, true, and aligned with Godâ€™s will (1 Thessalonians 5:21; Philippians 4:8).

By the end of the lesson, students will be able to:
