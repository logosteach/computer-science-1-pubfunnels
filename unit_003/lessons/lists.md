# Python Lists Video Series: Topic Outline

### Recommended Video Order

1. **Introduction to Python Lists**
   - What is a list?
   - How to create lists (empty and with values)
   - Lists vs other data types (strings, numbers)

2. **Accessing List Elements**
   - Indexing (positive and negative indexing)
   - Slicing lists
   - Common errors (`IndexError`)

3. **Modifying Lists**
   - Changing elements by index
   - Adding elements: `append()`, `insert()`, `extend()`
   - Removing elements: `remove()`, `pop()`, `del`, `clear()`

4. **List Operations and Built-in Functions**
   - Length (`len()`), membership (`in`)
   - Concatenation (`+`) and repetition (`*`)
   - `min()`, `max()`, `sum()` (with numbers)

5. **Iterating Over Lists**
   - `for` loops with lists
   - Using `range()` and `enumerate()`
   - Basic list comprehensions

6. **Sorting and Reversing Lists**
   - `sort()` vs `sorted()`
   - `reverse()` vs `reversed()`
   - Custom sorting with `key=`

7. **Advanced List Features**
   - Nested lists (lists inside lists)
   - Copying lists (shallow vs deep copy)
   - Common list methods summary + best practices

### Bonus Video (Optional)

- **Working with Lists in Real Projects**
  - Practical examples (to-do list, shopping cart, data processing)
