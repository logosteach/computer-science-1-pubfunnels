Python Dictionaries

In this lesson we are going to focus our attention on the dictionary data structure and how it is different from the list, set, and tuple. We are going to learn how to create empty dictionaries, how to add key-value pairs to them, and how to access values using the keys. We will also explore some of the built-in methods that dictionaries have to offer, such as .keys(), .values(), .items(), and .update(). We will also look at basic looping through dictionaries. By the end of this lesson you will have a solid understanding of how to use dictionaries in Python and how they can be useful for storing and organizing data in your programs.

Objectives

  • Create and work with dictionaries in Python (key-value pairs).
  • Access, add, modify, and remove items in a dictionary.
  • Iterate through dictionary keys, values, and items.
  • Use common dictionary methods such as get(), keys(), values(), items(), and update().
  • Understand when dictionaries are the best data structure for a task.
  • Reflect on how dictionaries store information by meaningful keys, just as our heavenly father knows each of us by name and calls us according to His purpose. (Isaiah 43:1, John 10:3).
