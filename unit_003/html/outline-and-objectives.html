Unit 3 - Lists, Sets, Tuples, Dictionaries and Modules

This unit contains the following lessons:

  • Lesson 1: Modules and Import Statements
  • Lesson 2: Lists
  • Lesson 3: Sets
  • Lesson 4: Tuples
  • Lesson 5: Dictionaries

Lesson 1 Objectives

By the end of the lesson, students will be able to:

  • explain what modules are and why they are useful in Python.
  • use the import statement to bring in built-in and external modules.
  • import specific functions or attributes from a module using from ... import ....
  • understand the difference between import module and from module import function.
  • reflect on how modules show the value of community and working together, just as God has gifted different people with different abilities for the benefit of the whole body of Christ (1 Corinthians 12:4-7, 12-27).

Lesson 2 Objectives

By the end of the lesson, students will be able to:

  • create and work with lists in Python.
  • access, modify, and add elements to a list using indexing and methods like append(), insert(), and remove().
  • use built-in functions such as len(), min(), max(), and sum() with lists.
  • iterate through a list using a for loop.
  • recognize that just as lists allow us to organize and work with collections of data, God gathers His people into communities and families where each person has value and purpose (Romans 12:4-5; Ephesians 4:16).

Lesson 3 Objectives

By the end of the lesson, students will be able to:

  • create and use sets in Python.
  • understand that sets automatically remove duplicates and are unordered.
  • perform common set operations such as union, intersection, difference, and symmetric difference.
  • use sets for efficient membership testing and removing duplicates from data.
  • reflect on how sets remind us of the uniqueness of each believer while still belonging to the same body — distinct yet united in Christ (1 Corinthians 12:12-13; Galatians 3:28).

Lesson 4 Objectives

By the end of the lesson, students will be able to:

  • create and work with tuples in Python.
  • understand the key difference between lists and tuples (tuples are immutable).
  • explain when to use a tuple instead of a list.
  • unpack tuples and use them for fixed collections of data.
  • recognize that tuples reflect the idea of immutable promises — once established, they do not change, just as God’s covenants and promises in Christ are secure and unchanging (Hebrews 6:17-18; 2 Corinthians 1:20).

Lesson 5 Objectives

By the end of the lesson, students will be able to:

  • create and work with dictionaries in Python (key-value pairs).
  • access, add, modify, and remove items in a dictionary.
  • iterate through dictionary keys, values, and items.
  • use common dictionary methods such as get(), keys(), values(), and items().
  • understand when dictionaries are the best data structure for a task.
  • reflect on how dictionaries store information by meaningful keys, just as God knows each of us by name and calls us according to His purpose (Isaiah 43:1; John 10:3).
