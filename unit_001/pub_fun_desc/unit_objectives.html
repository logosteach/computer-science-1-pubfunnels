Python Fundamentals - Unit Objectives

Review these learning objectives before beginning each lesson in the unit.

Lesson 001 - Downloading and Installing Python

  • Explain why Python is a popular programming language for beginners and professionals.
  • Download and install the latest version of Python from the official Python website (or through an OS-specific store).
  • Verify that Python was installed correctly by checking its version in a command line interface.
  • Launch and navigate the IDLE (Integrated Development and Learning Environment) shell.
  • Describe the basic layout and purpose of the IDLE interface.
  • Articulate how the careful process of downloading Python only from official, trustworthy sources parallels the biblical truth that "faith comes from hearing" (Romans 10:17).

Lesson 002 - Arithmetic, Variable Naming Rules and Conventions

  • Open and use the Python IDLE interactive shell.
  • Execute simple Python commands and expressions directly in the shell.
  • Understand the difference between the interactive shell and running saved scripts.
  • Use basic arithmetic operators and see immediate results in the shell.
  • Recognize and correct simple syntax errors in the interactive environment.
  • Reflect on how the interactive nature of the Python IDLE shell parallels our active relationship with God through worship, prayer, and obedience to His Word.

Lesson 003 - The Magnificent Print Function

  • Explain the purpose and importance of the print() function in Python.
  • Use the print() function to output text, numbers, and combinations of both.
  • Use escape sequences such as \n (newline) and \t (tab) within print() statements.
  • Concatenate strings and numbers using the + operator or commas in print().
  • Create simple, formatted output messages using the print() function.
  • Explain the importance of clear and intentional output in programming and draw a parallel to letting our speech and actions be gracious, clear, and purposeful (Colossians 4:6).

Lesson 004 - IDLE Script Editor

  • Open the IDLE script editor and create a new Python file.
  • Write, save, and run multi-line Python scripts using the IDLE editor.
  • Understand the difference between running code in the shell versus running a saved script.
  • Use comments (#) effectively to document their code.
  • Edit, debug, and re-run scripts within the IDLE editor.
  • Think deeply about building lasting relationships with other believers and being doers of the Word, not hearers only (James 1:22-25).

Lesson 005 - VS Code - Downloading, Installing & Features

  • Download and install Visual Studio Code (VS Code).
  • Install the Official Python extension for VS Code.
  • Create and open a new Python file in VS Code.
  • Write, save, and run Python scripts using VS Code’s integrated terminal and run button.
  • Compare the features of IDLE versus VS Code and explain when each tool is useful.
  • Understand the VS Code Marketplace and how to identify safe, reputable extensions.
  • Reflect on how different tools serve different purposes, just as God equips each believer with unique gifts for His kingdom work (1 Corinthians 12:4-7).
