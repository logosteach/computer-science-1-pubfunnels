Downloading and Installing Python

In this lesson we are going to download and install the Python programming language (and a nice little editor named IDLE that comes along with it) onto our personal computer system. Make sure you have the necessary permissions to install software on your machine, and follow the steps carefully to ensure a successful installation.

Lesson Objectives

  • Download and install the Python programming language directly from www.python.org

    • Activities:

      • The student will be given the web address https://www.python.org and give instructions on how to download the Python installer onto their personal computer.

      • The student will install Python onto their personal computer (an in addition with this installation comes the default editor IDLE)

    • Success Criteria: The student has downloaded and installed Python onto their personal computer and the application loads with no errors.

Concepts to think about

Getting Ready: What we are about to learn

Example 1: Chris opens his web-browser and clicks on a link he received in his email. The link is: www.python-install.com. A page loads that looks legit. Are there any concerns that Chris should have?

Yes! The official site for downloading and installing Python is www.python.org. Never use a link from an email unless it is from a trusted sources and you have read through the url link first. Changing links slightly and sending your to pages that look authentic is a primary tactic that online villains use to get your personal data.

Example 2: Danielle has clicked on the download link and her browser says that Python is downloading onto her system. However, it is 3 hours into the download. What could be going wrong?

It could be that Danielle has a slower connection, the download server that provides the software is down or experiencing higher traffic (unlikely), or that there is a connection problem with the server. Danielle should check her home router to make sure her connection is working correctly.

The site: https://www.speedtest.net is a legitimate site she can use to test her connection. Why don't you try it now!?
